The models behind the dashboards.

A dashboard without a data model is just a pretty chart. These pages show the capture → enrich → score → route → feedback pipeline for each deliverable — how raw clicks become qualified opportunities, and how the feedback loop gets closed. Interactive scoring calculators where it makes sense.
The canonical stackReused across all models
Capture: HubSpot, GA4, ad platforms, product events  •  Enrich: Clearbit / ZoomInfo / internal joins  •  Score: rules + model (HubSpot or Python)  •  Route: Salesforce / HubSpot workflows  •  Feedback: closed-won + disqual outcomes piped back to scoring.
